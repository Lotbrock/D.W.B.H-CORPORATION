Visualizar
Steps
1. El actor ingresa a la pestaña personal
2. SYSTEM Mostrar interfaz de usuario
3. El actor visualiza los usuarios registrados en el sistema
4. SYSTEM Muestra los usurios ingresado

Modificar
Steps
1. El actor modifica los permisos de los usuarios
2. SYSTEM mostrara un formulario con los datos del usuario
3. El actor diligenciara la informacion correspondiente
4. El actor guarda la informacion y publica
5. SYSTEM Guarda y publica los permisos y sesiones

Consultar
Steps
1. El actor cunsulta los usuarios registrados en el sistema
2. SYSTEM Muestra los usuarios requeridos por el actor
