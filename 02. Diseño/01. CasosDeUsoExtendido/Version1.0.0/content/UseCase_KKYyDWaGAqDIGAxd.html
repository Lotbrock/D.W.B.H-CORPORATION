Crear
Steps
1. Lider da click a "crear competencia"
2. SYSTEM Muestra un formulario con todos los datos necesarios para crear la competencia.
3. Lider le da click a "guardar"
4. SYSTEM Envia los nuevos datos a un nuevo registro en la base de datos.

Modificar
Steps
1. Lider da click a "modificar"
2. SYSTEM muestra los datos de la competencia ya seleccionada
3. Lider da click en "guardar"
4. SYSTEM reemplaza los datos en la base de datos
