Crear
Steps
1. El actor ingresa al módulo de versiones horarios.
2. SYSTEM Muestra una interfaz donde el actor podrá crear, modificar y consultar.
3. El actor elige la opción de crear.
4. SYSTEM Muestra una interfaz donde hay que completar los siguientes campos: número, versión, trimestre vigente, estado.
5. El actor completara los campos.
6. SYSTEM Muestra una interfaz, donde hay un botón para guardar los datos.
7. El actor dará clic en "Guardar".
8. SYSTEM Guardará los cambios.

Modificar
Steps
1. El actor ingresa al módulo de versiones horarios.
2. SYSTEM Muestra una interfaz donde el actor podrá crear, modificar y consultar.
3. El actor elige la opción de modificar.
4. SYSTEM Muestra una interfaz donde están almacenados datos guardados y podrá modificar los siguientes campos: número, versión, trimestre vigente, estado.
5. El actor modificara los campos que desee.
6. SYSTEM Muestra una interfaz, donde hay un botón para guardar los datos.
7. El actor dará clic en "Guardar".
8. SYSTEM Guardará los cambios.

Consultar
Steps
1. El actor ingresa al módulo de versiones horarios.
2. SYSTEM Muestra una interfaz donde el actor podrá crear, modificar y consultar.
3. El actor elige la opción de consultar.
4. SYSTEM Muestra una interfaz donde hay datos almacenados y allí podrá consultar los horarios que desee ya sea por: número, versión, trimestre vigente, estado.
5. El actor elegirá entre (número, versión, trimestre vigente, estado) el horario que desea buscar y dará clic en "Buscar".
6. SYSTEM Busca en la base de datos.
7. El actor espera que se carguen los datos que busco.
8. SYSTEM Muestra los datos.
