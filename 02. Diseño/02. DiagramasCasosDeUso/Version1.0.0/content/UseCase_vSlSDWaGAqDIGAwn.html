Modificar
Steps
1. elLider le da clic al boton de modificar
2. SYSTEM muestra los datos con la opcion de editarlos
3. Lider le edita lo que sea necesario y le da aceptar
4. SYSTEM Envia los nuevos datos a la base de datos y los reemplaza

Crear
Steps
1. Lider da click en el boton crear del modulo
2. SYSTEM muestra un formulario para llenar con todos los datos necesarios
3. Lider Llena el formulario y le da a crear
4. SYSTEM Envia los datos a la base de datos y lo guarda como un nuevo registro

Cambiar estado
Steps
1. Lider da click a cambiar estado
2. SYSTEM Muestra los estados disponibles para la competencia
3. Lider selecciona el estado que desea para la competencia y le da a guardar
4. SYSTEM Envia el dato de la disponibilidad y lo modifica en la base de datos
